# Оновлення

Те, як ви оновлюєте OpenChamber, залежить від того, як ви його встановили. У будь-якому разі ви можете перевірити свою поточну версію в **Settings → OpenChamber → About**.

## Десктопний застосунок

Десктопний застосунок перевіряє наявність оновлень у релізах GitHub. Коли воно доступне, OpenChamber повідомляє вам, завантажує його, коли ви оберете, і встановлює при наступному перезапуску. Ви завжди керуєте процесом — нічого не встановлюється без вашого схвалення.

## Веб / CLI

Якщо ви встановили CLI, оновіть його з **About** кнопками перевірки та оновлення або з терміналу:

```bash
openchamber update
```

OpenChamber визначає, як його було встановлено (npm, pnpm, yarn чи bun), і запускає правильне оновлення за вас.

## Сервер Codex

OpenChamber та Codex оновлюються окремо. Коли доступна нова версія Codex, OpenChamber пропонує оновити її й перезапускає сервер після цього. Див. [Codex Runtime](/uk/codex-server/) про те, як цим сервером керують.
